# Експериментални Протоколи за Индиректна Детекция

## Въведение

Тъй като **директната детекция на v > c феномени е невъзможна** поради EM затвора, всички експерименти трябва да търсят **индиректни сигнатури** - аномалии, несъответствия или корелации, които не могат да се обяснят в рамките на стандартната физика.

---

# Експеримент 1: Търсене на Енергийни Аномалии в LHC

## Цел

Детекция на систематична липсваща енергия при високоенергийни сблъсъци, индикираща преход към I_X пространства.

## Теоретична База

### Хипотеза

При E > E_threshold, част от енергията преминава в I_X канали:

```
E_total = E_EM_detected + E_X_missing

E_X_missing/E_total ≈ f(E/E_threshold)
```

### Очаквана Сигнатура

```
При E < E_threshold: ΔE_missing ≈ 0 (в рамките на грешката)
При E > E_threshold: ΔE_missing > 0 (систематично)
```

## Експериментален Протокол

### Фаза 1: Селекция на Събития

**Критерии:**

- Енергия на сблъсък: E_cm > 10 TeV
- Чиста топология (минимален фон)
- Добре реконструирани частици

**Очакван брой събития:**

- LHC Run 3 (2022-2025): ~10⁹ сблъсъка
- След селекция: ~10⁶ събития

### Фаза 2: Енергиен Баланс

**Измервания:**

```
E_in = 2 × E_beam (известно прецизно)
E_out = ∑ᵢ E_i (детектирани частици)
E_neutrino = E_missing_transverse (стандартен метод)

ΔE_anomaly = E_in - E_out - E_neutrino
```

**Очаквана точност:**

- E_in: δE/E ~ 10⁻⁴
- E_out: δE/E ~ 1% (калориметри)
- E_neutrino: δE/E ~ 5% (индиректно)

### Фаза 3: Статистически Анализ

**Метод:**

1. Разделяне на събития по енергия: E₁ < E₂ < ... < Eₙ
2. За всяка група: изчисляване на <ΔE_anomaly>
3. Търсене на систематична тенденция с E

**Очаквана сигнатура:**

```
<ΔE_anomaly>(E) = A × (E/E_threshold)ⁿ  при E > E_threshold
                = 0                     при E < E_threshold
```

**Статистически критерий:**

- Значимост > 5σ (стандарт за откриване)
- Не обяснима с известни процеси (неутрино, детекторни ефекти)

### Фаза 4: Систематични Грешки

**Контрол:**

- Калибрация на калориметри (различни методи)
- Симулации на неутрино процеси (Monte Carlo)
- Сравнение между различни детектори (ATLAS vs CMS)

**Очаквани систематични грешки:**

- Калориметрия: ~0.5%
- Неутрино модели: ~2%
- Общо: ~2.5%

### Фаза 5: Интерпретация

**Ако ΔE_anomaly > 3σ:**

- Проверка за нови частици (суперсиметрия, тъмна материя)
- Ако не се обяснява → кандидат за I_X преход

**Ако ΔE_anomaly < 3σ:**

- Поставяне на лимити: E_threshold > E_limit
- Планиране на бъдещи експерименти (FCC при по-високи енергии)

## Времева Линия

- **2025-2026:** Анализ на Run 3 данни
- **2027-2028:** Публикуване на резултати
- **2029+:** Планиране на FCC (ако има индикации)

## Очаквани Резултати

### Сценарий А: Положителен Резултат

```
ΔE_anomaly = (0.5 ± 0.1)% при E > 15 TeV
Значимост: 5.2σ
```

**Интерпретация:** Силна индикация за I_X преход при E_threshold ≈ 15 TeV

### Сценарий Б: Отрицателен Резултат

```
ΔE_anomaly = (0.05 ± 0.08)% (съвместимо с 0)
```

**Интерпретация:** E_threshold > 20 TeV (лимит)

# Експеримент 2: Космологични Корелации в CMB

## Цел

Детекция на корелации в космичния микровълнов фон (CMB) на ъглови мащаби извън светлинния хоризонт.

## Теоретична База

### Хипотеза

Ако в ранната Вселена I*X беше активен (Vmax >> c), регионите могат да са били причинно свързани на разстояния > c × t*Вселена.

### Очаквана Сигнатура

Корелационна функция C(θ) при θ > θ_horizon:

```
θ_horizon ≈ 1° (стандартна космология)

Стандартно: C(θ > θ_horizon) ≈ 0
С I_X: C(θ > θ_horizon) ≠ 0 (малка, но ненулева)
```

## Експериментален Протокол

### Фаза 1: Данни

**Източник:**

- Planck satellite (2009-2013): пълно небе, резолюция ~5'
- Бъдещи мисии: LiteBIRD, CMB-S4 (по-висока чувствителност)

**Данни:**

- Температурна карта: T(θ, φ)
- Поляризация: Q(θ, φ), U(θ, φ)

### Фаза 2: Изчисляване на Корелационна Функция

**Метод:**

```
C(θ) = <T(n₁) T(n₂)>  където |n₁ - n₂| = θ

Разлагане на сферични хармоници:
T(θ, φ) = ∑ₗₘ aₗₘ Yₗₘ(θ, φ)

Спектър на мощността:
Cₗ = <|aₗₘ|²>
```

**Фокус:**

- Ниски l (l < 10) → големи ъглови мащаби (θ > 10°)
- Търсене на аномалии в Cₗ

### Фаза 3: Статистически Анализ

**Стандартна космология предсказва:**

```
Cₗ (l < 10) → определени стойности (от инфлация)
```

**Търсене на отклонения:**

```
ΔCₗ = Cₗ_observed - Cₗ_theory

Ако ΔCₗ > 3σ → аномалия
```

**Известни аномалии (Planck):**

- "Axis of Evil" (предпочитана посока)
- "Cold Spot" (необичайно студена зона)
- Ниска квадруполна амплитуда (l = 2)

**Нова интерпретация:**
Може ли да се обяснят с I_X корелации?

### Фаза 4: Моделиране

**Модел с I_X:**

```
В ранна Вселена (t < t_transition):
  Vmax = Vmax_X >> c
  Корелационна дължина: ξ = Vmax_X × t

След преход (t > t_transition):
  Vmax = c
  Корелационна дължина: ξ = c × t
```

**Предсказание:**

```
Cₗ_I_X = Cₗ_standard + ΔCₗ(Vmax_X/c, t_transition)
```

**Фитване на параметри:**

- Vmax_X/c (колко пъти по-бързо)
- t_transition (кога е станал преходът)

### Фаза 5: Сравнение с Алтернативи

**Други обяснения:**

- Инфлационни модели (различни потенциали)
- Топологични дефекти (космични струни)
- Първични гравитационни вълни

**Дискриминация:**

- I_X модел прави специфични предсказания за поляризация
- Различна зависимост от l

## Времева Линия

- **2025-2027:** Анализ на Planck данни с I_X модел
- **2028-2030:** LiteBIRD данни (по-висока чувствителност)
- **2031+:** CMB-S4 (окончателен тест)

## Очаквани Резултати

### Сценарий А: Положителен Резултат

```
ΔCₗ (l = 2-5) = (5 ± 1.5) μK² (3.3σ)
Най-добър фит: Vmax_X/c ≈ 100, t_transition ≈ 10⁻³⁵ s
```

**Интерпретация:** Силна индикация за I_X в ранната Вселена

### Сценарий Б: Отрицателен Резултат

```
ΔCₗ съвместимо с 0 (< 2σ)
```

**Интерпретация:** Ако I_X съществува, Vmax_X/c < 10 или t_transition < 10⁻³⁶ s

# Експеримент 3: Прецизна Спектроскопия на Квазари (VSL Тест)

## Цел

Търсене на вариация в "константата" c чрез спектроскопия на далечни квазари.

## Теоретична База

### Хипотеза

Ако вакуумът е имал различна "плътност" в миналото, Vmax може да е била различна:

```
Vmax(z) = c × [1 + α(z)]

където z е redshift (мярка за време в миналото)
```

### Очаквана Сигнатура

Промяна в фина структурна константа:

```
α_EM = e²/(4πε₀ℏc)

Ако c варира: α_EM(z) = α_EM(0) × c(0)/c(z)
```

**Измерване:**
Спектрални линии на атоми в квазари (z ~ 2-4):

```
Δλ/λ ∝ Δα_EM/α_EM
```

## Експериментален Протокол

### Фаза 1: Селекция на Квазари

**Критерии:**

- Висок redshift: z > 2 (далечни, стари)
- Ярки: m < 18 (достатъчно светлина за спектроскопия)
- Богати на метални линии (Fe II, Mg II, и т.н.)

**Очакван брой:**

- SDSS каталог: ~10⁵ квазари
- След селекция: ~1000 подходящи

### Фаза 2: Спектроскопия

**Инструменти:**

- VLT/UVES (резолюция R ~ 50,000)
- Keck/HIRES (R ~ 70,000)
- Бъдещи: ELT/HIRES (R ~ 100,000)

**Измервания:**

- Позиции на спектрални линии (λ_obs)
- Сравнение с лабораторни стойности (λ_lab)

### Фаза 3: Анализ на Дублети

**Метод:**
Различни преходи имат различна чувствителност към α_EM:

```
Δω/ω = q × Δα_EM/α_EM

където q е коефициент на чувствителност (различен за всяка линия)
```

**Пример:**

- Fe II λ1608: q ≈ +0.5
- Mg II λ2796: q ≈ 0

**Сравнение:**

```
Δα_EM/α_EM = (Δω_Fe - Δω_Mg) / (q_Fe - q_Mg)
```

### Фаза 4: Статистически Анализ

**За всеки квазар:**

- Изчисляване на Δα_EM/α_EM
- Грешка: δ(Δα_EM/α_EM) ~ 10⁻⁶ (текуща точност)

**Комбиниране на данни:**

```
<Δα_EM/α_EM>(z) = средно от всички квазари при redshift z
```

**Търсене на тенденция:**

```
Δα_EM/α_EM = A × z + B × z²  (полиномиален фит)
```

### Фаза 5: Систематични Грешки

**Контрол:**

- Изотопни ефекти (различни изотопи на Fe, Mg)
- Кинематични ефекти (движение на газ в квазара)
- Инструментални ефекти (калибрация на спектрограф)

**Очаквани систематични:**

- Изотопи: ~5 × 10⁻⁷
- Кинематика: ~10⁻⁶
- Инструментални: ~5 × 10⁻⁷

## Времева Линия

- **2025-2028:** Наблюдения с VLT, Keck
- **2029-2032:** Анализ и публикуване
- **2033+:** ELT наблюдения (по-висока точност)

## Очаквани Резултати

### Сценарий А: Положителен Резултат

```
Δα_EM/α_EM = (-5 ± 1.5) × 10⁻⁶ при z ~ 3
Значимост: 3.3σ
```

**Интерпретация:** c беше по-голяма в миналото (Vmax_past > c_now)?

### Сценарий Б: Отрицателен Резултат

```
Δα_EM/α_EM = (0.2 ± 0.8) × 10⁻⁶ (съвместимо с 0)
```

**Интерпретация:** c е константа до 10⁻⁶ за последните 12 милиарда години

# Експеримент 4: Мултимесенджър Астрономия (Предвестници)

## Цел

Търсене на сигнали, които пристигат **преди** електромагнитните сигнали от астрофизични събития.

## Теоретична База

### Хипотеза

Ако I_X съществува с Vmax > c, сигнали в I_X ще пристигат преди EM сигнали:

```
t_X = d/Vmax_X
t_EM = d/c

Δt = t_EM - t_X = d × (1/c - 1/Vmax_X)

При Vmax_X >> c: Δt ≈ d/c
```

### Очаквана Сигнатура

**За сливане на неутронни звезди при d = 100 Mpc:**

```
Δt ≈ 100 Mpc / c ≈ 300 милиона години × (1/c) ≈ 10 секунди
```

**НО:** Детекцията на I_X сигнал е проблематична (EM затвор!)

**Алтернатива:** Търсене на **индиректни ефекти** на ранното пристигане

## Експериментален Протокол

### Фаза 1: Мониторинг

**Инструменти:**

- LIGO/Virgo/KAGRA (гравитационни вълни)
- IceCube (неутрино)
- Fermi, Swift (гама-лъчи)
- Оптични телескопи (видима светлина)

**Стратегия:**

- Непрекъснат мониторинг на небето
- Автоматични алерти при детекция
- Бърза последваща наблюдения (< 1 минута)

### Фаза 2: Корелационен Анализ

**При детекция на събитие:**

1. Записване на точното време на всеки сигнал
2. Изчисляване на времеви разлики: Δt*GW-EM, Δt*ν-EM
3. Сравнение с теоретични предсказания

**Търсене на аномалии:**

```
Стандартно: Δt_GW-EM ≈ 0 (в рамките на грешката)
С I_X: Δt_GW-EM < 0 (GW пристига преди EM?)
```

### Фаза 3: Анализ на Детекторен Шум

**Идея:**
Ако I_X сигнал пристига преди EM, може да има **предварителни ефекти** в детекторите.

**Метод:**

- Анализ на шума в LIGO 10-60 секунди **преди** EM детекция
- Търсене на корелации с последващото събитие
- Статистическо сравнение с фонов шум

**Очаквана сигнатура:**

```
Аномално увеличение на шум в специфична честотна лента
Корелирано с енергията на последващото събитие
```

### Фаза 4: Неутрино Предвестници

**Специфичен случай:**
Неутрино могат да избягат от ядрото на колапсираща звезда преди светлината.

**Пример - SN 1987A:**

```
Неутрино детектирани: 23:07:35 UTC
Оптична светлина: ~3 часа по-късно
```

**Нов тест:**
При бъдеща близка суперно
ва (d < 10 kpc):

- Очакваме неутрино ~10 секунди преди светлина (стандартно)
- Ако има I_X: може да има допълнителни предвестници?

## Времева Линия

- **2025-2030:** Непрекъснат мониторинг, ~10 събития/година
- **2031+:** Анализ на натрупани данни

## Очаквани Резултати

### Сценарий А: Положителен Резултат

```
Събитие GW170817-подобно:
Δt_GW-EM = -2.3 ± 0.5 секунди (GW преди EM!)
Значимост: 4.6σ
```

**Интерпретация:** Силна индикация за v_GW_real > c

### Сценарий Б: Отрицателен Резултат

```
Δt_GW-EM = +0.1 ± 0.3 секунди (съвместимо с 0)
```

**Интерпретация:** v_GW = c (в рамките на грешката) или I_X ефектите са много малки

# Експеримент 5: Търсене на Вакуумна Дисперсия

## Цел

Детекция на енергийно-зависима скорост на светлината, индикираща дискретна структура на вакуума.

## Теоретична База

### Хипотеза

Ако вакуумът има дискретна структура (квантова гравитация), дисперсионната релация се модифицира:

```
E² = p²c² + m²c⁴ + α(E/E_Planck)ⁿ p²c²

Ефективна скорост:
v_eff = dE/dp ≈ c [1 + α(E/E_Planck)ⁿ]
```

### Очаквана Сигнатура

Фотони с различни енергии пристигат в различно време:

```
Δt = d × (v₁⁻¹ - v₂⁻¹) ≈ d/c × α(E₁ⁿ - E₂ⁿ)/E_Planckⁿ
```

## Експериментален Протокол

### Фаза 1: Селекция на Източници

**Критерии:**

- Далечни GRB (Gamma-Ray Bursts): d > 1 Gpc
- Кратка продължителност: Δt_intrinsic < 1 секунда
- Широк енергиен спектър: 0.1 keV - 100 GeV

**Очакван брой:**

- Fermi GBM: ~300 GRB/година
- След селекция: ~10 подходящи/година

### Фаза 2: Измерване на Времена на Пристигане

**Инструменти:**

- Fermi LAT (0.1-300 GeV)
- Swift XRT (0.3-10 keV)
- Времева резолюция: ~1 μs

**Метод:**
За всеки фотон:

- Енергия: E_i
- Време на пристигане: t_i

### Фаза 3: Статистически Анализ

**Корелация E vs t:**

```
t(E) = t₀ + d/c + Δt_dispersion(E)

Фит: Δt_dispersion(E) = A × Eⁿ
```

**Очаквано:**

```
Стандартно: A = 0 (няма дисперсия)
С квантова гравитация: A ≠ 0
```

### Фаза 4: Текущи Лимити

**От GRB наблюдения:**

```
|Δt/t| < 10⁻¹⁷ при E ~ 100 GeV, d ~ 10 Gpc

Лимит на E_Planck: E_Planck > 10¹⁸ GeV (съвместимо със стандартно)
```

**Бъдещи възможности:**

- CTA (Cherenkov Telescope Array): по-висока чувствителност
- По-далечни източници (z > 5)

## Времева Линия

- **2025-2030:** Анализ на Fermi данни
- **2031+:** CTA наблюдения

## Очаквани Резултати

### Сценарий А: Положителен Резултат

```
A = (5 ± 1.2) × 10⁻¹⁸ s/GeV
Значимост: 4.2σ
```

**Интерпретация:** Вакуумът има дискретна структура при E_Planck

### Сценарий Б: Отрицателен Резултат

```
A = (0.3 ± 0.8) × 10⁻¹⁸ s/GeV (съвместимо с 0)
```

**Интерпретация:** Структурата е по-фина от E_Planck или няма дисперсия

- Поставяне на строги лимити
